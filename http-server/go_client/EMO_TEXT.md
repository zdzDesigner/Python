

 积极情感：
   - 喜悦：开心、愉快、兴奋、欣喜、满足
   - 爱：热爱、喜欢、迷恋、钟情、宠爱
   - 感激：感谢、感恩、感动、温暖
   - 希望：期待、憧憬、乐观、信心

  消极情感：
   - 悲伤：难过、痛苦、失落、沮丧、绝望
   - 愤怒：生气、愤怒、恼火、愤慨、暴怒
   - 恐惧：害怕、担心、焦虑、恐惧、恐慌
   - 厌恶：讨厌、反感、恶心、嫌弃

  中性/复杂情感：
   - 惊讶：震惊、意外、诧异、惊喜
   - 困惑：迷茫、疑惑、纠结、矛盾
   - 平静：淡定、从容、宁静、安详
   - 怀旧：怀念、思念、惆怅、感伤

  强度描述：
   - 轻微：有点、稍微、些许
   - 强烈：极度、非常、特别、万分
   - 持续：一直、持续、久久、长期


  叙事类：
   - 平静客观：淡然、从容、平稳
   - 温暖亲切：温和、柔和、贴心
   - 深沉厚重：庄重、深沉、凝重

  情感类：
   - 悲伤忧郁：哀伤、忧郁、惆怅
   - 激昂澎湃：激昂、澎湃、振奋
   - 神秘悬疑：神秘、诡异、紧张

  文艺类：
   - 诗意浪漫：唯美、浪漫、优雅
   - 怀旧感伤：怀念、感慨、时光流转
   - 哲理思考：深邃、思辨、内省

  实用建议：
   - 纪录片：客观平静
   - 爱情故事：温暖柔和
   - 悬疑剧情：神秘紧张
   - 历史题材：庄重深沉
   - 广告宣传：积极振奋
